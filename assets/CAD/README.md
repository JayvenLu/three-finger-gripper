# Three-Finger Gripper CAD

This directory contains the editable STEP design sources and derived GLB files used for interactive browser previews.

## Files

| Source file | Description | Derived preview |
| --- | --- | --- |
| `3finger_gripper.STEP` | Complete three-finger gripper assembly | `previews/3finger_gripper.glb` |
| `mold.STEP` | Tactile sensor mold assembly | `previews/mold.glb` |

Use the STEP files as the preferred source for manufacturing, inspection, and modification. GLB files are tessellated derivatives and may not preserve exact CAD geometry.
